### SDSC Summer Institute 2026
# Session 2.4 Code Migration & Software Environments

**Date:** Monday, August 3, 2026

**Time:** 3:30 PM - 5:00 PM Pacific

**Summary:** This session introduces key strategies for transitioning computations to HPC systems, including using pre-installed software, compiling code with appropriate tools, and setting up Python and conda environments. It also covers workflow management and introduces containerized solutions with Singularity, featuring hands-on practice on SDSC.

**Presented by:** [Yuwu Chen](https://www.sdsc.edu/education/on-demand-learning/202512-R-for-HPC.html) ychen64 @ucsd.edu

**Hands-On:** Commands for simple hands-on work (can be done asynchronous w/ talk) are in Code-Migration-Handson.md file.
